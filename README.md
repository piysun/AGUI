# AI Data Analyst

An intelligent AI-powered data analysis application that allows users to query databases, visualize data, and build interactive dashboards using natural language.

Built with **LangGraph**, **CopilotKit**, and **Next.js**.

---

## 🚀 Features

- 🤖 **AI-Powered Data Analysis**
  - Ask questions in natural language
  - Automatically generates and executes SQL queries

- 📊 **Rich Data Visualizations**
  - Supports:
    - Bar charts
    - Pie charts
    - Line charts
    - Area charts
    - Scatter plots
    - Stacked bar charts
    - Tables

- 📌 **Live Dashboard**
  - Pin visualizations
  - Auto-refresh with live data updates

- 💬 **Thread-Based Conversations**
  - Multiple analysis sessions
  - Persistent chat history

- 🔄 **Real-Time Updates**
  - Refresh intervals:
    - 30 seconds
    - 1 minute
    - 5 minutes
    - Manual refresh

- 🎨 **Modern UI**
  - Built with Next.js & Tailwind CSS
  - Fully responsive design

- 🗄️ **PostgreSQL Integration**
  - Direct DB connectivity via MCP (Model Context Protocol)

---

## 🏗️ Tech Stack

- **Frontend:** Next.js, Tailwind CSS
- **AI Framework:** LangGraph, CopilotKit
- **Database:** PostgreSQL
- **Protocol:** MCP (Model Context Protocol)
